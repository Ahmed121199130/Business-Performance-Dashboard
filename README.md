## üåü Project Overview

This project showcases a comprehensive, interactive Business Performance Dashboard developed entirely using **Power BI Desktop**. The primary goal is to provide deep, actionable insights into sales operations and supply chain efficiency across multiple functional areas.

The dashboard uses a structured **Dark Theme** design for optimal readability and data contrast.

## üéØ Key Analytical Features

The dashboard is structured across **seven dedicated pages** to provide specialized analysis:

1.  **Executive Summary:** Top-level KPIs (Total Sales, Orders, Freight, AOV).
2.  **Customers Analysis:** Top N customers by sales and order volume.
3.  **Categories Analysis:** Sales distribution and volume by product category.
4.  **Orders Analysis:** Time-based trends and shipping/logistics efficiency.
5.  **Product Analysis:** Inventory distribution and unit price metrics.
6.  **Employees Analysis:** Sales performance and effectiveness of employees.
7.  **Suppliers Analysis:** Financial contribution and geographic distribution of suppliers.

## ‚ú® Core Functionality & Design

* **Unified Dark Theme:** Professional aesthetic ensuring high contrast for data visualization.
* **Global Filters:** Features four synchronized, global slicers (Date, Country, Category, Product) that filter data across all seven pages simultaneously.
* **Reset Mechanism:** Dedicated "Clear all slicers" button for quick data reset.
* **Advanced Visuals:** Utilization of Bar Charts for Top N analysis, Scatter Charts for efficiency metrics, and Treemaps for inventory/volume distribution.
* **Intuitive Navigation:** Dedicated side navigation panel for seamless movement between analysis pages.
